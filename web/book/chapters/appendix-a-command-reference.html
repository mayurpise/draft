Intent Mapping

Draft recognizes natural language equivalents for all commands. Saying "set up the project" triggers /draft:init. "Hunt bugs" triggers /draft:bughunt. "What's the status?" triggers /draft:status. See the Draft overview (/draft) for the full intent mapping table.

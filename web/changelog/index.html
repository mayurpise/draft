Changelog

Release history and updates for Draft.

v2.4 2026-04-26 Latest
Code Intelligence
  • Knowledge graph engine — tree-sitter parsers for Go, Python, TS/JS, C/C++ + ctags fallback for Java/Rust/Ruby/Swift. Six query modes: callers, impact, hotspots, modules, cycles, mermaid.
  • Confidence-tagged call edges — every call edge marked direct (bare identifier) or inferred (member call) so review/bughunt can weight findings.
  • Atomic incremental builds via per-module SHA-256 hashing — only changed modules re-extract.
  • Track impact memory in metadata.json — each completed track records files touched, modules affected, downstream count, by-category breakdown. /draft:new-track surfaces overlap warnings during context load.
Tooling
  • 14 deterministic shell helpers under scripts/tools/ — git metadata, file classification, hotspot ranking, cycle detection, freshness checks, ADR indexing, test-framework detection, more. Uniform JSON output and exit-code contract; graceful degradation.
  • 16 new tool tests bringing the suite to 25 total.
  • make build and make lint Makefile targets added.
Methodology
  • Refreshed all 28 skills with deeper guidance and "Red Flags — STOP if you're..." preambles.
  • architecture.md template expanded from 25 sections to 28 sections + 5 appendices with tier-based output minimums for large codebases.
  • New shared procedures: graph-query.md, parallel-analysis.md (Map/Reduce IR — ~60% wall-clock cut at XL tier).
Fixes
  • Hardened Copilot syntax transform: kebab-case skill names only, email-shaped tokens preserved.
  • TS module-edge resolution correctly resolves multi-segment relative imports.
  • JSON escape helper strips ASCII control characters.
  • Glob exclude patterns anchored to full-string match.
  • 16 stale TOC anchors fixed in methodology.md.
